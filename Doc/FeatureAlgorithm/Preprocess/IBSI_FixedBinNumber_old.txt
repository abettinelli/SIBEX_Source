-Description: 
This method is to scale the image intensity into the certain bit range. 

-Parameters:
1. BinNumber:  Bit depth for the scaled image
2. RangeMin:   Mininum value of the source range.
3. DynMin:     If RangeMin == 1 RangeMin is dynamically determined by min of the current image.
4. RangeMax:   Maximum value of the source range.
3. DynMax:     If RangeMax == 1 RangeMax is dynamically determined by max of the current image.

-Formula:
InputRange=[Param.RangeMin, Param.RangeMax];
FinalRange=[1, Param.BinNumber];
CurrentData=(CurrentData-InputRange(1))*(FinalRange(2)-FinalRange(1))/(InputRange(2)-InputRange(1))+FinalRange(1);
